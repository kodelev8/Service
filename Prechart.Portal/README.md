# Angular & Jest
## Project setup
```
npm install
```

### Updates packages
npm update
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npx jest
```

### Update your snapshots
```
npx jest --updateSnapshot
```

### Lints and fixes files
```
npm run lint
```
